# Learning Library Curriculum

## Purpose

The Learning Library is a structured educational program designed to build deep understanding through expandable lessons.

Each lesson teaches a core concept clearly while allowing unlimited future expansion.

The curriculum is organized so that later topics build naturally upon earlier ones.

---

# Ultimate Goal

Develop the mathematical, physical, and engineering knowledge required to understand Maxwell's Equations and other advanced topics from first principles.

---
