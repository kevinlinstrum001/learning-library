# START HERE

If you're returning to this project after a break, follow these steps.

## 1. Read the project specification

Read:

- PROJECT_SPEC.md

This explains the overall structure of the project.

---

## 2. Determine your task

Choose one:

### Build a new lesson

Upload:

- config/ruleset.json
- templates/lesson-template.html

Then describe the lesson you want to create.

### Expand an existing lesson

Upload:

- the lesson HTML
- config/ruleset.json

Then specify which section you want to expand.

### Modify the authoring system

Upload:

- config/ruleset.json

Describe the change you want.

Review the proposed changes.

Approve them.

Replace the updated files.

---

## 3. Check the changelog

If you're unsure what's changed since your last session, read:

docs/CHANGELOG.md

## Updating the Authoring System

1. Upload the current `ruleset.json`.
2. Describe the desired changes.
3. Review the proposed update.
4. Replace `config/ruleset.json`.
5. Update `docs/CHANGELOG.md`.
6. Update any affected project files listed in the change summary.
